Diagram downloads

Three Excalidraw source files accompany the Candles v2 design (parent page: Market Data Engine — Candles v2). They live in the market-data-engine repo at docs/diagrams/ and render in any Excalidraw client.

How to view / export

  1. Open excalidraw.com (or the VS Code Excalidraw plugin).

  2. File → Open → pick the .excalidraw file from your local clone of market-data-engine.

  3. Export as PNG / SVG from the same menu if you need a static image.

No git clone needed if you only want to view — grab the raw file from Bitbucket:
